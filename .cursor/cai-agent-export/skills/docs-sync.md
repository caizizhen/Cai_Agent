## 技能：文档同步更新（Docs Sync）

> 适用场景：代码或规则/技能新增后，确保 README（`README.md` / `README.zh-CN.md`）、架构文档、变更日志（`CHANGELOG.md` / `CHANGELOG.zh-CN.md`）同步。

### 目标

- 保证「实现、文档、变更记录」三者一致，降低认知偏差。

### 典型步骤

1. 列出本次变更点；
2. 同步更新 `README.md`（英文默认）与 `README.zh-CN.md`（中文）中的能力说明或目录现状；
3. 同步更新 `CHANGELOG.md`（英文默认）与 `CHANGELOG.zh-CN.md`（中文）的版本条目；
4. 检查是否还需更新架构文档或示例配置。
