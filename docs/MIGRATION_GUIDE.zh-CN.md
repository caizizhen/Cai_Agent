# 迁移指南

> 英文原文见 [`MIGRATION_GUIDE.md`](MIGRATION_GUIDE.md)。本文件是中文伴随摘要，用于保持当前文档集的中英文入口完整。

## 用途

`MIGRATION_GUIDE.md` 记录版本升级时可能影响配置、JSON 输出、脚本解析或 CI 接入的变化。升级前应先阅读英文原文，再结合根目录 `CHANGELOG.md` / `CHANGELOG.zh-CN.md` 检查当前版本差异。

## 维护规则

- 英文原文仍是迁移细节的主要来源。
- 如果新增破坏性变更、配置迁移或 JSON schema 变更，需要同步更新本摘要和英文原文。
- 不在本文件复制完整迁移表，避免与英文原文漂移。
