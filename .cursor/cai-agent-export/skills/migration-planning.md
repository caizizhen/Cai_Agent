## 技能：迁移方案设计（Migration Planning）

> 适用场景：需要对配置结构、模块布局、命令接口或底层实现做渐进式迁移。

### 一、目标

- 设计低风险、可分阶段回退的迁移方案，避免一次性大爆炸改动。

### 二、对话提示模版

> 「请帮我为以下变更设计迁移方案：\<迁移目标\>。  
> 需要说明：现状、目标状态、分阶段步骤、兼容策略、验证方法和回滚方案。」

### 三、典型步骤

1. 说明现状与迁移动机；
2. 划分目标状态与中间状态；
3. 设计兼容层、过渡配置或双写/双读策略；
4. 为每一阶段制定验证与回滚方法。

### 四、注意事项

- 迁移重点是控制风险，不是追求一步到位；
- 涉及用户可见接口时，必须同步更新文档与变更记录。
