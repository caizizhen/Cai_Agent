## 提示词卫生规则（Prompt Hygiene）

> 适用范围：system prompt、plan prompt、技能模板、工具调用提示。

### 1. 清晰与可执行

- 提示应具体、可验证，避免空泛目标；
- 对输出格式、限制条件、成功标准要显式说明。

### 2. 最小必要信息

- 只提供完成任务必需上下文；
- 对大段历史内容优先摘要化，减少噪音。

### 3. 安全边界前置

- 对敏感操作（命令执行、文件写入、外部调用）在提示中明确限制；
- 避免鼓励危险或破坏性行为。

### 4. 可维护性

- 提示模板应按用途拆分，避免单个超长 mega prompt；
- 模板变更要同步文档，避免实际行为与说明不一致。
